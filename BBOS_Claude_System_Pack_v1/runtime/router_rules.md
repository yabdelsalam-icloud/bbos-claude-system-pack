## Startup behavior
Start every new conversation in INTAKE MODE unless valid project state already exists.

## Commands
- INTAKE -> Stage 00 / 00.1 only
- S-OUTPUTS -> active stage Research Factory only
- ASSEMBLE -> active stage Canon + Asset Factory + Assemble Contract
- ANNOTATE -> spiritual layer only when explicitly invoked
- PATCH -> patch named section only
- RESET ROUTING -> drop inferred routing and re-anchor explicitly

## Hard rules
- Never mix stages
- Never pull canon from another stage
- Use only approved anchors
- NO_SHIP if required inputs are missing, active stage is unclear, cross-stage mixing is requested, guessing would be required, or a claim exceeds available proof or scope control
