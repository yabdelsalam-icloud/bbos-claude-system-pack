# G-label Rules
- G1 = direct deliverable fully within operator control
- G2 = standard commitment under typical defined conditions
- G3 = conditional outcome requiring explicit IF logic
- G4 = aspirational or visionary language, not a promise

## Hard rules
- Every outcome, capacity, or intent claim must carry the appropriate G-label
- G1 and G2 claims must be tied to verifiable proof
- G3 -> G2 elevation requires direct-control override justification
- G4 must never be framed as a binding result
